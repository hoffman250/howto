Basic Map

Getting a basic map up and running

Getting started with embedding a basic map with fixed latitude and longitude coordinates is almost straight forward. Google provides a javascript example that allows the programmer to drop in some code into their html doc and have a map centered on their lat/long coordinates embedded on their webpage.

Google provides a code example similar to the code below.

function initMap() { var map = new google.maps.Map(document.getElementById("googleMap1"), { center: {lat: 44.050, lng: -121.300}, zoom:7, mapTypeId:google.maps.MapTypeId.ROADMAP }); }

Unfortunately, I could not get the above code to work as is. After doing some research on the internet, I found a code example here:

w3schools Basic Map Example

That added this line of code:

google.maps.event.addDomListener(window, 'load', initMap);

This code runs the function "initMap" once the DOM is loaded. Without this line of code, apparently initMap() would never run. As shown below, a basic google map now loads up fine, with this code added. In this example, I have the map defaulting to center on Bend, Oregon, my hometown.
