API Key?

Basic Maps Require No API Key

What Google does not come right and tell you is that you can use their API without getting an API key. Apparently so long as you are not a heavy user of the API, Google permits a limited number of interactions with the API per day. This is great news for somebody like me that wants to go learn how to interact with this API without going through the hassle of signing up for an API key and then worrying about security of said key.

For professionals that have higher volume interactions with Google Maps, they can get a key right from the Google Developers Console. Instructions for getting an API key can be found here:

Google Maps Javascript API

With the standard key, Google permits up to 25,000 map loads per day for free. Beyond that, they start billing.
